02. 개발진

이진원 (팀장)
  • CSR(React) + SSR(Thymeleaf) 혼합 아키텍처 설계 및 공지/소개 페이지 SSR 적용
  • 회원 관리 / 등급 변경 / 정지 처리 / 상세 활동 내역 조회 기능 전체 설계 및 구현
  • 회원별 게시글/댓글/투표 활동 데이터를 통합 조회하는 복합 SQL 작성 (여러 테이블 JOIN + UNION ALL)
  • 게시판 신고/좋아요/활동 통계 데이터를 집계하기 위한 GROUP BY 기반 집계 SQL 구현
  • 관리자 검색/정렬/필터 UI를 위한 MyBatis 동적 SQL 구조 설계
  • 다중 테이블 데이터 조회 시, 서브쿼리 구조 최적화 및 ROWNUM 기반 페이징 처리 경험
이용재
  • 카테고리별 게시글 목록 출력
  • 카테고리별 게시글 작성 기능 구현
  • 자유 게시판 CRUD 구현
서정민
  • 회원 탈퇴 기능 구현
  • 회원가입 정규 표현식 구현
  • 비밀번호, 닉네임, 이메일 구현
  • 회원 정보 수정 구현
전현우
  • 투표 기능 및 쪽지 기능 개발
